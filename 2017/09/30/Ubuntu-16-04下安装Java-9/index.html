1 下载JDK

去JAVA官网上把自己对应自己版本jdk下好 ( 附下载链接
http://www.oracle.com/technetwork/java/javase/downloads/jdk9-downloads-3848520.html)

2 解压JDK

打开终端，先cd到你要想安装java的位置，比如说我是安装到/home/huweicai/software目录下，因此使用cd命令：
cd /home/huweicai/software
然后使用
sudo tar -zxvf /home/huweicai/下载/jdk-8u131-linux-x64.tar.gz
我大概解释一下这条命令：sudo是使用用户权限，tar是解压和压缩的一个指令，-zxvf则是对应的参数，而后面这一串是jdk的文件位置

3 配置环境变量

在终端输入：
sudo gedit /etc/profile
（就相当于在windos下用记事本打开配置文件） 在末端加上（第一行后面为你解压的jdk路径）
export JAVA_HOME=/home/huweicai/software/jdk1.8.0_131
export JRE_HOME=${JAVA_HOME}/jre
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib
export PATH=${JAVA_HOME}/bin:$PATH
（第一行后面为你解压的jdk路径）source /etc/profile 我把这个单列出来因为当初在这里卡了特别特别久，后来才发现原来我修改完之后还没有生效，还要让系统运行一下那个文件，因此需要在终端中输入:source /etc/profile 才算配置好了环境变量5 测试jdk是否安装成功

4 检验是否安装成功

在终端中输入java -verison，如果出来如下输出的话，那就说明你配置成功了，Congradulations，jdk就安装好了 ，如果失败的话，自己想办法吧
